👁️基础检查：视力和眼位

📋 视功能检查的起点

1.1 视力检查

  • 远视力：5米距离，Snellen视力表或LogMAR表
  • 近视力：40cm，近用视力表（如Jaeger表）
  • 矫正视力：屈光矫正后测量的最佳视力
  • 裸眼视力：未矫正状态下的视力

⚠️ 记录规范

右眼 OD / 左眼 OS / 双眼 OU

如：OD 1.0 / OS 0.8 / OU 1.0

或使用小数：0.1-1.5

或使用LogMAR：0.0 ~ 1.0

📊Morgan双眼视功能正常值

🎯 临床诊断的金标准参考

📌 什么是Morgan正常值？

1949年，Australian Optometrist Joan Morgan 通过大样本研究，建立了正常人群的双眼视功能参数参考范围，至今仍是临床诊断的重要标准。

检查项目正常值范围异常提示
隐斜（远）1Δ外隐斜 ± 2Δ≥4Δ内/外隐斜需关注
隐斜（近）3Δ外隐斜 ± 3Δ内隐斜>5Δ提示集合过度
集合近点（NPC）≤5cm / ≥7cm>7cm提示集合不足
融像性集合（远）BI/BOsX/6±2 / 9±4 ΔBI破裂点<4Δ提示发散不足
融像性集合（近）BI/BOs13±4 / 21±3 Δ近距BO不足提示集合过度
调节幅度（最小）15 - 年龄/4（D）低于公式值提示调节不足
NRA（负相对调节）+2.00 ± 0.50 D>+2.50提示过度放松
PRA（正相对调节）-2.37 ± 1.00 D>-3.00提示调节储备充足
调节灵敏度（单眼）≥12cpm<10cpm提示调节异常
调节灵敏度（双眼）≥8-10cpm双眼低于单眼提示双眼问题
AC/A比值3:1 ~ 5:1 Δ/D高AC/A>6:1提示集合过度

💡 记忆技巧：Hofstetter调节幅度公式

最小调节幅度 = 15 - 年龄 ÷ 4

例如：40岁人群，最小调节幅度 = 15 - 40÷4 = 5D（相当于20cm）

如果测量值远低于此，提示调节不足，需要训练。

⚡调节功能检查

🔧 调节检查四件套

调节幅度 → NRA/PRA → 调节灵敏度 → BCC/AMP

1. 调节幅度（AMP）

  • 移近法：40cm视标逐渐移近，直到模糊
  • 计算：调节幅度 = 1/模糊距离（m）
  • 如：10cm模糊 → 1/0.1 = 10D

2. 相对调节（NRA/PRA）

  • NRA：在近视距加正镜，直到持续模糊
  • PRA：在近视距加负镜，直到持续模糊
  • 意义：评估调节的储备和放松能力

3. 调节灵敏度（Flipper）

  • 设备：反转拍（±2.00D常用）
  • 方法：看40cm视标，+2.00/-2.00交替
  • 记录：每分钟完成多少周期（cpm）
  • 单眼：主要测调节，双眼：测调节+融像

4. 调节反应（BCC/MEM）

  • 内容：评估调节的准确性（超前/滞后）
  • 正常：+0.25D ~ +0.75D（轻微滞后）
  • 超前：<0.00D → 调节过度
  • 滞后：>+1.00D → 调节不足

AC/A比值 梯度法或计算法（远隐斜-近隐斜）/（远调节-近调节） 3:1 ~ 5:1 Δ/D >6:1高AC/A：集合过度
<3:1低AC/A：集合不足

🔑 融像检查三个关键点

模糊点（Blur）：融像储备开始使用（调节性集合参与）

破裂点（Break）：融像崩溃，出现复视或抑制

恢复点（Recovery）：减少棱镜后重新恢复融像

记录格式：Blur / Break / Recovery，如 X/10/6

👀眼球运动检查

🎯 注视、扫视、追随三大功能

类型功能检查方法异常表现
注视稳定性维持视线稳定观察眼静态时的微小运动；使用 visuoscope检查注视类型注视微扫视增多、漂移
扫视运动快速跳跃式移动视线观察从一视标到另一视标的准确性；NSUCO扫视测试欠冲/过冲、精确度下降
追随运动平滑跟踪移动目标跟随移动视标（如笔灯），观察有无跳动出现catch-up saccade（补充扫视）

简化临床评估：阅读相关眼动

  • 阅读跳行测试：让孩子读一段文字，观察跳行、漏字、重复的情况
  • 字母表扫视：快速阅读随机排列的字母，评估扫视效率和准确性
  • NR眨眼测试：正常阅读时的眨眼频率（约15-20次/分钟）
📋视功能检查标准化流程
问诊 + 主诉 → 视力检查 → 屈光检查
眼位检查 → 调节功能 → 集合功能 → 眼球运动
立体视检查 → 数据整理 → 诊断分析

检查注意事项

① 顺序很重要：先单眼后双眼，先静止后动态

② 全程屈光矫正：有屈光不正者需配镜后检查

③ 远近都要查：远距和近距的检查结果可能完全不同

④ 双眼检查不可缺：单眼正常≠双眼功能正常

⑤ 结合主诉：检查数据必须与患者症状对照分析
