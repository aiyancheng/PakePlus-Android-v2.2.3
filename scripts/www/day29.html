典型案例1：弱视+斜视+异常视网膜对应

案例信息：

女，6岁。家长主诉：发现右眼"有点斜"3年余。检查：睫状肌麻痹验光：右眼+4.50D（0.4），左眼+2.00D（1.0）；角膜映光：右眼内斜15Δ；同视机：主客观斜视角均=+15°，无正常视网膜对应（主副角差<5°但有抑制）；遮盖试验：右眼旁中心注视。

诊断：

  1. 屈光参差性弱视（右眼）+ 斜视性弱视（混合机制）
  2. 部分调节性内斜视（睫状肌麻痹后仍有残余内斜）
  3. 旁中心注视 + 异常视网膜对应

训练方案：

  1. 阶段一（1-4周）：屈光全矫配镜+遮盖（左眼遮盖2小时/天）+精细训练（穿珠子、描图，每天15分钟）
  2. 阶段二（5-8周）：注视性质纠正——海丁格刷+后像疗法（改善旁中心注视）
  3. 阶段三（9-16周）：遮盖维持+精细训练加量+双眼视基础建立
  4. 阶段四（>4个月）：评估是否需要斜视手术（残余内斜角度>10Δ→手术指征）；术后进行融像训练
  5. 复查：每2周视力，每4周全面复查

典型案例2：调节过度+集合不足+视疲劳

案例信息：

男，22岁，程序员。主诉：每天看电脑8小时后头痛、眼酸胀、视物模糊3个月余。检查：AMP=14D（正常），NRA=+1.25D（↓），PRA=-1.50D（↓），MEM=-0.50D（超前），AF=单眼15cpm/双眼7cpm（↓）；近距隐斜8Δ外（↑），NPC=9cm（↓），AC/A=2.5/1（低）；CISS=22分（异常）。

诊断：

  1. 调节过度（睫状肌痉挛）——MEM超前，NRA低
  2. 集合不足（低AC/A型）——近距外斜、NPC远移
  3. 视疲劳综合征（视频终端综合征）

训练方案：

  1. 优先级：先放松调节（改善过度）→再加强集合
  2. 第1-4次（调节放松阶段）：远距字母表训练（每天15分钟）+负镜片快速阅读（每次10分钟）；减少近距用眼强度
  3. 第5-8次（集合提升阶段）：聚散球10min/天+裂隙尺BO训练+反转拍（±1.50D，较低难度）
  4. 第9-12次（综合整合）：Hart字母表+近距工作任务模拟训练
  5. 近附加：考虑+0.75D近附加减少电脑使用时的调节负担

典型案例3：眼球运动异常+阅读障碍

案例信息：

男，9岁，三年级。家长主诉：阅读速度慢、跳行漏字、语文成绩差，老师反映注意力不集中。检查：视力正常（1.0/1.0）；眼位正常；DEM测试：水平时间120秒（异常，正常<47秒），错误率20次（异常）；NSUCO追随评分偏低；集合/调节检查正常。

诊断：

  1. 扫视功能异常（发育性扫视迟缓）
  2. 学习相关视觉效率低下（眼球运动相关）
  3. （需排除：注意缺陷多动障碍、学习障碍等其他原因——建议转心理评估）

训练方案：

  1. 第1-4次：Dotting O's（圈O训练）+吸管追踪训练——注视稳定性
  2. 第5-8次：Hart字母表扫视训练（从大字→小字，距离从3m→5m）+数字快速定位
  3. 第9-12次：马氏登球追随训练+近距阅读任务模拟
  4. 家庭配合：每天字母表训练5分钟；家长监督
  5. 复查：每4周复查DEM测试数据

🛠️ 实操任务：独立完成3例复杂案例解决方案

要求（分组进行，每组完成以下内容）：

  1. 完整诊断（含主次诊断）
  2. 训练方案（含阶段划分、工具选择、频次时长）
  3. 复查计划（含评估指标）
  4. 预估疗程和预后

评估标准：诊断准确率60%+，方案合理性40%。

必须终止训练的情况

  • 出现新的眼部器质性疾病
  • 患者依从性极差，无法配合
  • 原诊断有误，需重新全面评估

📝 今日考核

  1. 案例1中，该6岁内斜视合并弱视患者，为何先做弱视治疗再做斜视手术？
  2. 案例2中，为什么要先"放松调节"再"加强集合"？
  3. 案例3中，如果集合和调节检查正常，是否就不需要做视觉训练？为什么？
  4. 复杂案例分析：你认为以上3个案例中，哪个预后最好？哪个最难？为什么？
参考答案：
1. 儿童发育期先提升弱视眼视力、纠正注视性质，术后才能建立双眼视，否则易复发；
2. 调节痉挛未放松时，集合训练会加重疲劳，必须先解除调节紧张；
3. 仍需训练，因为阅读障碍核心是眼球运动功能异常，与调节集合无关；
4. 预后最好：案例3（眼球运动）；最难：案例1（弱视+斜视+旁中心注视）。
