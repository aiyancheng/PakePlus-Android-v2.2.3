🎯 今日学习目标

① 掌握消除抑制的3种核心方法；② 熟练融像训练的标准化操作；③ 掌握立体视检查和训练方法；④ 能独立操作「脱抑制+融像」训练全流程。

🔓一、消除抑制训练

抑制——双眼视功能的第一杀手

抑制是大脑主动忽略来自一只眼的信息以避免复视。必须先消除抑制，才能进行融像训练。

方法1：Worth四点灯

  1. 被检者配红绿眼镜（右红左绿）
  2. 注视40cm处四点灯仪（2红+2绿）
  3. 正常：4灯（红绿融合）
  4. 抑制：仅2灯（健眼）或3灯（差眼）
  5. 训练时引导患者"主动放松，用双眼同时看"

方法2：Bagolini线状镜

  1. 被检者戴Bagolini透镜（条纹方向互相垂直）
  2. 注视点光源
  3. 正常双眼视：看到完整交叉线（两对角线）
  4. 抑制：仅一条线或一条完整+一条断裂

方法3：后像训练

  1. 强光照射后产生后像（正后像/负后像）
  2. 利用后像引导双眼使用
  3. 适用于旁中心注视抑制
🔗二、融像训练

融像——双眼视的核心

裂隙尺（Stereo Slit）——最常用融像工具

  1. 将裂隙尺水平放置，被检者透过裂隙观看
  2. 集合训练（BO方向）：将裂隙向鼻侧移动，要求双眼融像
  3. 发散训练（BI方向）：将裂隙向颞侧移动，要求双眼融像
  4. 记录破裂点（开始出现模糊/复视）和恢复点
  5. 训练参数：每次10-15个循环，每天2-3组
  6. 考核要求：能独立完成BO和BI方向操作，记录数据

偏心圆卡（Eccentric Circles）

  1. 将卡片置于40cm，调整双眼视线
  2. 注视融合的图像，逐渐移向鼻侧/颞侧
  3. 目标：周边融像→中心融像

矢量图（Vectograms）

  1. 使用偏振分离双眼图像
  2. 通过滑动控制图像分离程度
  3. 精确训练融像范围和立体视
👁️三、立体视检查与训练

立体视——双眼视的最高形式

检查方法操作正常值
Titmus立体图偏振眼镜下观察立体图，说出图案≤40秒弧
随机点立体图（RDS）随机点中找到凸起/凹陷图形≤60秒弧
Brewster立体镜使用Brewster立体镜融像测量≤60秒弧

🛠️ 实操考核（课堂分组进行）

考核项目：

  1. 独立操作裂隙尺BO（集合）和BI（发散）训练——记录破裂点和恢复点
  2. 完成Worth四点灯检测——判断有无抑制
  3. 完成Titmus立体视检测——记录最佳立体视锐度
  4. 设计一套完整的"脱抑制+融像"训练流程（口头演示）

📝 今日考核

  1. Worth四点灯检查结果的3种情况（4灯/2灯/3灯）各代表什么含义？
  2. 裂隙尺BO和BI方向分别训练什么融像能力？
  3. 立体视正常值是多少？ Titmus和RDS各适合什么场景？
  4. 复述"脱抑制→融像→立体视"三步训练流程。
📌核心知识点总结

双眼视核心训练金标准

  • 训练顺序：先脱抑制（Worth四点灯/Bagolini镜）→ 再融像 → 最后立体视
  • 融像工具：裂隙尺（BO集合/BI发散）、偏心圆卡、矢量图为核心工具
  • 立体视标准：Titmus≤40秒弧，随机点立体图≤60秒弧为正常
  • 实操关键：标准化操作，精准记录破裂点、恢复点、立体视锐度
