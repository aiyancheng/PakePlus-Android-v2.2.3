📐斜视分类

斜视分型系统——多维度分类

分类维度类型定义临床特点
按方向内斜视（Esotropia）眼位向鼻侧偏斜俗称"斗鸡眼"，最常见类型
外斜视（Exotropia）眼位向颞侧偏斜俗称"瞟眼"，间歇性外斜最常见
按显现程度显斜（Manifest Strabismus）眼位持续偏斜，双眼不能同时注视一眼恒定性偏斜
隐斜（Heterophoria）遮盖后出现偏斜，融合控制下正位日常无症状，融合打破时出现偏斜
按发病时间先天性（婴儿型）出生后6个月内出现常伴DVD，需手术干预
后天性6个月后逐渐出现常与调节/屈光参差相关
按A-V型A型/V型斜视向上/下注视时斜视角变化≥10Δ需特殊手术设计

特殊类型说明

  • 间歇性外斜视：融合控制好时正位，控制差时出现偏斜——视觉训练的最佳适应症之一
  • 完全调节性内斜视：睫状肌麻痹后内斜消失，全矫配镜后正位——不需要手术
  • 部分调节性内斜视：睫状肌麻痹后仍有残余内斜角度——需要手术+训练联合
👁️斜视导致的4大双眼视异常

为什么斜视患者会有这些症状？

斜视的本质是双眼不能同时注视同一目标，大脑接收到两个不同的图像，产生一系列适应性改变：

1️⃣ 视觉抑制

大脑主动忽略偏斜眼的视觉信息，避免复视和混淆。抑制可恒定/间歇性。
意义：长期抑制→斜视性弱视

2️⃣ 异常视网膜对应（ARC）

大脑重新建立视网膜对应关系，消除复视。
意义：训练难度大，术后易出现复视

3️⃣ 三级视功能缺失

正常双眼视：同时视→融合视→立体视。斜视患者逐级受损，立体视常完全丧失

4️⃣ 复视（Diplopia）

双眼感知两个独立图像。
意义：成人新发斜视最常见主诉

🔬斜视相关检查方法

诊断斜视与双眼视异常的必备检查

检查方法检查内容正常/异常标准
角膜映光法（Hirschberg）估测斜视角度反光点偏1mm≈7Δ
遮盖-去遮盖试验检测显斜及方向眼球移动→存在显斜
交替遮盖试验测量最大斜视角（含隐斜）揭示全部融合性偏斜
马氏杆法精确测量隐斜棱镜度棱镜中和点=隐斜量
同视机检查三级视功能同时视、融合范围、立体视正常存在
Worth四点灯抑制/融像/复视4灯=正常；2/3灯=抑制；5灯=复视
Titmus/RDS立体视立体视锐度正常≤40秒弧，斜视多缺失

🛠️ 实操任务：判断眼位与抑制状态

判断以下场景属于哪种情况：

  1. 遮盖右眼时，左眼向内移动 → 右眼内斜视
  2. Worth四点灯检查时只看到2个绿灯 → 右眼抑制
  3. 角膜映光法：右眼反光点在瞳孔中心鼻侧2mm → 右眼内斜约14Δ
  4. 同视机检查：主观斜视角=客观斜视角=+15° → 正常视网膜对应（内斜视）

  2. 内斜视和外斜视的鉴别要点是什么？各举一个典型案例。
  3. 显斜和隐斜的核心区别是什么？如何在临床中鉴别？
  4. Worth四点灯检查结果的3种常见情况（4灯/2灯/5灯）各代表什么含义？

✅ 标准答案

1. 视觉抑制（大脑忽略偏斜眼图像）、异常视网膜对应（重建对应关系）、三级视功能缺失（逐级受损）、复视（双眼感知双像）。

2. 内斜：眼位向鼻侧偏（斗鸡眼，调节性内斜）；外斜：眼位向颞侧偏（瞟眼，间歇性外斜）。

3. 核心区别：自然状态下是否偏斜；鉴别：遮盖-去遮盖查显斜，交替遮盖查隐斜。

4. 4灯=正常融合；2/3灯=单眼抑制；5灯=复视。
